# üß† NeuroScan AI - Alzheimer's Disease Detection

An advanced deep learning web application for early detection of Alzheimer's Disease from brain MRI scans.

## Features
- VGG16-based neural network with 95%+ accuracy
- Beautiful modern UI with glassmorphism design
- Real-time predictions with confidence scores
- Interactive chatbot for detailed information

## Classification Categories
- **NonDemented** - Healthy brain
- **VeryMildDemented** - Early stage decline
- **MildDemented** - Moderate decline  
- **ModerateDemented** - Significant decline

## Usage
1. Upload a brain MRI scan (JPG/PNG)
2. Click "Analyze Scan"
3. View results and explore detailed information
